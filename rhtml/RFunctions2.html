Miscellaneous Functions

Download RFunctions2.R (GitHub)

  • groupSum computes and displays a summary of a numeric vector v, optionally by grouped by g. Missing group IDs are treated as a separate group.

    
groupSum(v, g=NULL, Combined=TRUE, select_stats=NULL)

  • editRedCapData cleans the data.frame created by REDCap's .r file.

    
editRedCapData(d, changeNames=TRUE)

  • multTime multiplies a time value by multiplier. This function answers questions such as:
    "If I run 4:12/km for 42.195km, how long will it take?
    The lap argument can be specified in "HH:MM:SS" or "MM:SS" format (character string) or MM.SS format (numeric).

    
multTime(lap="4:12", multiplier=42.195)
multTime(lap=4.12, multiplier=42.195)

  • addTime sums time values from the vector times, which can be specified in "HH:MM:SS" or "MM:SS" format (character string) or MM.SS format (numeric). The outputs of multTime and addTime can be used as inputs for this function. For example:
    "If I run 4:02/km for 5km, then 3:55/km for 7km, what is the total time?

    
addTime( c(multTime("4:02",5), multTime(3.55,7)))
